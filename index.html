Project 2: Digital Dice

A Single Chip Electronic Dice created using a 4060 IC and 555 timer that generates a randomized output between 1 to 6 with the help of clock pulses generated by the timer. The output of any one LED corresponding to the number is set to high while remaining outputs are set to low.
Concepts used in creating the dice:

  • Circuit Building using breadboard

  • Working of a 4060 IC

  • Working of a 555 timer
